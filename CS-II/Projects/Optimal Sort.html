Data Graph


Explanation of Graph: This graph is a representation between the average execution time in seconds to the type of sorting algoritm, as well as file size and if the file was ordered, reversed, or in a random order. From the graph it is apparent that there is a positive relationship between execution time and the size of a file. In addition, the files that were already ordered took, on average, less time to order.
